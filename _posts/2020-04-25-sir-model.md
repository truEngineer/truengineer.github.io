–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ $S$, $I$ –∏ $R$ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ª—é–¥–µ–π –≤ –∫–∞–∂–¥–æ–π –≥—Ä—É–ø–ø–µ –≤ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–µ –≤—Ä–µ–º—è $t$, —Ç.–µ. –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –ª—é–¥–µ–π –≤ –∫–∞–∂–¥–æ–π –≥—Ä—É–ø–ø–µ —Ä–∞—Å—Å—á–∏—Ç—ã–≤–∞–µ—Ç—Å—è –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å–≤–∏–∏ —Å —Ñ—É–Ω–∫—Ü–∏—è–º–∏ –≤—Ä–µ–º–µ–Ω–∏: $S(t)$, $I(t)$ –∏ $R(t)$. –ú–æ–¥–µ–ª—å *SIR* –≤ —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –Ω–µ–ª–∏–Ω–µ–π–Ω—É—é —Å–∏—Å—Ç–µ–º—É –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞–ª—å–Ω—ã—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π (–î–£), –æ–ø–∏—Å—ã–≤–∞—é—â–∏—Ö –¥–∏–Ω–∞–º–∏–∫—É —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—è –≤ –ø–æ–ø—É–ª—è—Ü–∏–∏:

$$
\begin{align}
& \frac{dS}{dt} = - \frac{\beta IS}{N}, \\
& \frac{dI}{dt} = \frac{\beta IS}{N} - \gamma I, \\
& \frac{dR}{dt} = \gamma I.
\end{align} 
$$

–ó–¥–µ—Å—å $S(t)$ ‚Äî —á–∏—Å–ª–µ–Ω–Ω–æ—Å—Ç—å –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –≤ –º–æ–º–µ–Ω—Ç –≤—Ä–µ–º–µ–Ω–∏ $t$;
$I(t)$ ‚Äî —á–∏—Å–ª–µ–Ω–Ω–æ—Å—Ç—å –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –≤ –º–æ–º–µ–Ω—Ç –≤—Ä–µ–º–µ–Ω–∏ $t$;
$R(t)$ ‚Äî —á–∏—Å–ª–µ–Ω–Ω–æ—Å—Ç—å –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –≤ –º–æ–º–µ–Ω—Ç –≤—Ä–µ–º–µ–Ω–∏ $t$;
$N$ ‚Äî —á–∏—Å–ª–µ–Ω–Ω–æ—Å—Ç—å –ø–æ–ø—É–ª—è—Ü–∏–∏;
$\beta$ ‚Äî –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç–∏ –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ –∏–Ω–¥–∏–≤–∏–¥–æ–≤ —Å –ø–æ—Å–ª–µ–¥—É—é—â–∏–º –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ–º;
$\gamma$ ‚Äî –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç–∏ –ø–µ—Ä–µ—Ö–æ–¥–∞ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –≤ –≥—Ä—É–ø–ø—É –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö.

–í –∞–Ω–≥–ª–æ—è–∑—ã—á–Ω–æ–π –ª–∏—Ç–µ—Ä–∞—Ç—É—Ä–µ –ø–∞—Ä–∞–º–µ—Ç—Ä $\beta>0$ –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–π —á–∞—Å—Ç–æ—Ç–æ–π –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ (*effective contact rate*). –í—ã—á–∏—Å–ª—è–µ—Ç—Å—è —ç—Ç–æ—Ç –ø–æ–∫–∞–∑–∞—Ç–µ–ª—å –∫–∞–∫ —Å—Ä–µ–¥–Ω–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ –æ–¥–Ω–æ–≥–æ –∏–Ω–¥–∏–≤–∏–¥–∞ —Å –¥—Ä—É–≥–∏–º–∏ –≤ –¥–µ–Ω—å $\bar{c}$, —É–º–Ω–æ–∂–µ–Ω–Ω–æ–µ –Ω–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∑–∞—Ä–∞–∂–µ–Ω–∏—è $\tau$ (*transmissibility*) –ø—Ä–∏ –∫–æ–Ω—Ç–∞–∫—Ç–µ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ –∏–Ω–¥–∏–≤–∏–¥–∞ —Å –Ω–µ–∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–º $\beta=\bar{c}\tau$. –í –ø—Ä–æ—Å—Ç–µ–π—à–µ–π –º–æ–¥–µ–ª–∏ SIR –æ–±—ã—á–Ω–æ –ø—Ä–µ–¥–ø–æ–ª–∞–≥–∞–µ—Ç—Å—è, —á—Ç–æ —ç—Ç–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ä–∞–≤–Ω–∞ –µ–¥–∏–Ω–∏—Ü–µ, —Ç.–µ. –ø—Ä–∏ –∫–æ–Ω—Ç–∞–∫—Ç–µ –∑–¥–æ—Ä–æ–≤–æ–≥–æ –∏–Ω–¥–∏–≤–∏–¥–∞ —Å –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–º –∑–∞—Ä–∞–∂–µ–Ω–∏–µ –ø–µ—Ä–≤–æ–≥–æ –Ω–µ–∏–∑–±–µ–∂–Ω–æ. –ü–∞—Ä–∞–º–µ—Ç—Ä $\beta$ –≤ —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏–∑—É–µ—Ç –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ –∏–Ω–¥–∏–≤–∏–¥–æ–≤ —Å –ø–æ—Å–ª–µ–¥—É—é—â–∏–º –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ–º. –¢–æ–≥–¥–∞ $\beta{IS/N}$ ‚Äî —ç—Ç–æ –ø—Ä–∏—Ä–æ—Å—Ç –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö (*rate of new infections*). –ò–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ $\beta$ —É–º–Ω–æ–∂–∞–µ—Ç—Å—è –Ω–∞ —Ç–µ–∫—É—â–µ–µ —á–∏—Å–ª–æ –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ $S$ –∏ –∑–∞—Ç–µ–º ‚Äî –Ω–∞ –¥–æ–ª—é –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤, –≤ –∫–æ—Ç–æ—Ä—ã—Ö —É—á–∞—Å—Ç–≤—É—é—Ç –∑–∞—Ä–∞–∂—ë–Ω–Ω—ã–µ $I/N$, –≤—Å—ë –ª–æ–≥–∏—á–Ω–æ.

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –ø–µ—Ä–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –≤ —Å–∏—Å—Ç–µ–º–µ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –∏–∑–º–µ–Ω–µ–Ω–∏–µ —á–∏—Å–ª–∞ –∑–¥–æ—Ä–æ–≤—ã—Ö, –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö –∫ –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—é –∏–Ω–¥–∏–≤–∏–¥–æ–≤ —É–º–µ–Ω—å—à–∞–µ—Ç—Å—è —Å–æ –≤—Ä–µ–º–µ–Ω–µ–º –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ —á–∏—Å–ª—É –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ —Å –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–º–∏ $-\beta{IS/N}$. –ü–æ—Å–ª–µ –∫–æ–Ω—Ç–∞–∫—Ç–∞ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –∑–∞—Ä–∞–∂–µ–Ω–∏–µ, –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã–π –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω–æ–≥–æ. –í—Ç–æ—Ä–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç, —á—Ç–æ —Å–∫–æ—Ä–æ—Å—Ç—å —É–≤–µ–ª–∏—á–µ–Ω–∏—è —á–∏—Å–ª–∞ –∑–∞—Ä–∞–∑–∏–≤—à–∏—Ö—Å—è —Ä–∞—Å—Ç–µ—Ç –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ —á–∏—Å–ª—É –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ –∑–¥–æ—Ä–æ–≤—ã—Ö –∏ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö $\beta{IS/N}$ –∏ —É–º–µ–Ω—å—à–∞–µ—Ç—Å—è –ø–æ –º–µ—Ä–µ –≤—ã–∑–¥–æ—Ä–æ–≤–ª–µ–Ω–∏—è –ø–æ—Å–ª–µ–¥–Ω–∏—Ö $-\gamma{I}$, —Ç—Ä–µ—Ç—å–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ ‚Äì —á—Ç–æ —á–∏—Å–ª–æ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö –≤ –µ–¥–∏–Ω–∏—Ü—É –≤—Ä–µ–º–µ–Ω–∏ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ —á–∏—Å–ª—É –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö $\gamma{I}$.

–ü–∞—Ä–∞–º–µ—Ç—Ä $0<\gamma<1$ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏–∑—É–µ—Ç –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –ø–µ—Ä–µ—Ö–æ–¥–∞ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –≤ –≥—Ä—É–ø–ø—É –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö (*removed*) –∏–ª–∏ –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –≤—ã–±—ã–≤–∞–Ω–∏—è –∏–Ω–¥–∏–≤–∏–¥–æ–≤ (*removal rate*), —Ç.–µ. –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –∫–∞–∫–∞—è –¥–æ–ª—è –æ—Ç —Ç–µ–∫—É—â–µ–≥–æ —á–∏—Å–ª–∞ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö.

–†–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏–µ –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—è –≤ –º–æ–¥–µ–ª–∏ *SIR* –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ –Ω–µ–∑–∞—Ç–µ–π–ª–∏–≤–æ–π —Å—Ö–µ–º–µ "–≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã–µ —Å—Ç–∞–Ω–æ–≤—è—Ç—Å—è –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–º–∏, –ø–æ—Ç–æ–º –ø–µ—Ä–µ—Ö–æ–¥—è—Ç –≤ —Ä–∞–∑—Ä—è–¥ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö":

–°—Ç—Ä–µ–ª–∫–∏ –Ω–∞ —Å—Ö–µ–º–µ –≤—ã—à–µ –ø–æ–º–µ—á–µ–Ω—ã —Å–∫–æ—Ä–æ—Å—Ç—è–º–∏ –ø–µ—Ä–µ—Ö–æ–¥–∞ –∏–Ω–¥–∏–≤–∏–¥–æ–≤ –º–µ–∂–¥—É –≥—Ä—É–ø–ø–∞–º–∏. 

–ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã $\gamma$ –∏ $\beta$ —Ç–∞–∫–∂–µ –º–æ–≥—É—Ç –±—ã—Ç—å –≤—ã—Ä–∞–∂–µ–Ω—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ —á–µ—Ä–µ–∑ —Å—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è –º–µ–∂–¥—É –∫–æ–Ω—Ç–∞–∫—Ç–∞–º–∏ $T_c$ –∏ —Å—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è, –∑–∞ –∫–æ—Ç–æ—Ä–æ–µ –±–æ–ª–µ–∑–Ω—å –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è —Å —Ç–µ–º –∏–ª–∏ –∏–Ω—ã–º –∏—Å—Ö–æ–¥–æ–º $T_r$. –ó–¥–µ—Å—å –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ –∑–∞ –≤—Ä–µ–º—è $T_r$ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–æ–∂–µ—Ç –∑–∞—Ä–∞–∂–∞—Ç—å –¥—Ä—É–≥–∏—Ö, –ø–æ—ç—Ç–æ–º—É $T_r$ –µ—â—ë –∏–Ω–æ–≥–¥–∞ –Ω–∞–∑—ã–≤–∞—é—Ç —Å—Ä–µ–¥–Ω–∏–º –≤—Ä–µ–º–µ–Ω–µ–º –∑–∞—Ä–∞–∑–Ω–æ—Å—Ç–∏ (*mean infectious time*):

$$
\begin{align}
& \beta = T_c^{-1}, \\
& \gamma = T_r^{-1}.
\end{align}
$$

–ï—Å–ª–∏ —Å–ª–æ–∂–∏—Ç—å –≤—Å–µ —Ç—Ä–∏ —É—Ä–∞–≤–Ω–µ–Ω–∏—è –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–π –≤—ã—à–µ —Å–∏—Å—Ç–µ–º—ã –î–£ *SIR*, —Ç–æ –ø–æ–ª—É—á–∏–º, —á—Ç–æ:

$$\frac{dS}{dt} + \frac{dI}{dt} + \frac{dR}{dt} = 0.$$

–¢.–µ. —á–∏—Å–ª–µ–Ω–Ω–æ—Å—Ç—å –ø–æ–ø—É–ª—è—Ü–∏–∏ ‚Äì —ç—Ç–æ –Ω–µ–∫–æ—Ç–æ—Ä–∞—è –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ $N$, –∫–æ—Ç–æ—Ä–∞—è –Ω–µ –º–µ–Ω—è–µ—Ç—Å—è –≤ —Ç–µ—á–µ–Ω–∏–µ —ç–ø–∏–¥–µ–º–∏–∏:

$$S(t) + I(t) + R(t) = N.$$

–í–º–µ—Å—Ç–æ –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è —Å–∏—Å—Ç–µ–º—ã –î–£ –æ–≥—Ä–∞–Ω–∏—á–∏–º—Å—è —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–∏–µ–º –ø–∞—Ä—ã –∑–∞–Ω–∏–º–∞—Ç–µ–ª—å–Ω—ã—Ö —Ñ–∞–∫—Ç–æ–≤. –í—Ç–æ—Ä–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å–∏—Å—Ç–µ–º—ã –º–æ–∂–Ω–æ –ø–µ—Ä–µ–ø–∏—Å–∞—Ç—å –≤ –≤–∏–¥–µ:

$$
\begin{align}
\frac{dI}{dt} & = \gamma{I} \left( \frac{\beta}{\gamma}\frac{S}{N} - 1 \right) = \\ 
& =  \gamma{I} \left( R_0\frac{S}{N} - 1 \right).
\end{align} 
$$

$$R_0 = \frac{\beta}{\gamma} = \frac{T_r}{T_c}.$$

–ü–∞—Ä–∞–º–µ—Ç—Ä—ã –º–æ–¥–µ–ª–∏ –±—É–¥—É—Ç —Å–ª–µ–¥—É—é—â–∏–º–∏: —Ä–∞–∑–º–µ—Ä –ø–æ–ø—É–ª—è—Ü–∏–∏ $N=2\cdot10^3$, –æ–¥–∏–Ω –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –≤ –Ω–∞—á–∞–ª–µ —ç–ø–∏–¥–µ–º–∏–∏ (–Ω—É–ª–µ–≤–æ–π –ø–∞—Ü–∏–µ–Ω—Ç), —Å—Ä–µ–¥–Ω–µ–µ —á–∏—Å–ª–æ –∫–æ–Ω—Ç–∞–∫—Ç–æ–≤ —Å –ø–æ—Å–ª–µ–¥—É—é—â–∏–º –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ–º –Ω–∞ –æ–¥–Ω–æ–≥–æ —á–µ–ª–æ–≤–µ–∫–∞ –≤ –¥–µ–Ω—å $\beta=1.5$ (–∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–æ–∂–µ—Ç –∑–∞—Ä–∞–∑–∏—Ç—å —Ç—Ä–µ—Ö –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö –∑–∞ –¥–≤–∞ –¥–Ω—è, —Å—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è –º–µ–∂–¥—É –∫–æ–Ω—Ç–∞–∫—Ç–∞–º–∏ $T_c=0.66$), –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –≤—ã–±—ã–≤–∞–Ω–∏—è –∏–Ω–¥–∏–≤–∏–¥–æ–≤ $\gamma=0.5$ (–∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–π –º–æ–∂–µ—Ç –∑–∞—Ä–∞–∂–∞—Ç—å –¥—Ä—É–≥–∏—Ö –≤ —Ç–µ—á–µ–Ω–∏–µ –¥–≤—É—Ö –¥–Ω–µ–π $T_r=2$). –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –±–∞–∑–æ–≤–æ–µ —Ä–µ–ø—Ä–æ–¥—É–∫—Ç–∏–≤–Ω–æ–µ —á–∏—Å–ª–æ $R_0={\beta}/{\gamma}=3$.

–†–µ—à–∞—Ç—å —Å–∏—Å—Ç–µ–º—É –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏–∞–ª—å–Ω—ã—Ö —É—Ä–∞–≤–Ω–µ–Ω–∏–π *SIR* –±—É–¥–µ–º —á–∏—Å–ª–µ–Ω–Ω–æ, –∏—Å–ø–æ–ª—å–∑—É—è —Å–≤—è–∑–∫—É –±–∏–±–ª–∏–æ—Ç–µ–∫ *NumPy+SciPy*. –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π IPython-–Ω–æ—É—Ç–±—É–∫ —Å —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–µ–π —ç—Ç–æ–≥–æ —á–∏—Å–ª–µ–Ω–Ω–æ–≥–æ —Ä–µ—à–µ–Ω–∏—è –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ [—Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–∏](https://github.com/truEngineer/epidemic-models). –¢–∞–º –º–æ–∂–Ω–æ –æ—Å—É—â–µ—Å—Ç–≤–∏—Ç—å —Ä–∞—Å—á—ë—Ç –∏ –¥–ª—è –¥—Ä—É–≥–∏—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –º–æ–¥–µ–ª–∏. –ó–¥–µ—Å—å –∂–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –≥—Ä–∞—Ñ–∏–∫–∏ —Ä–∞–∑–≤–∏—Ç–∏—è —ç–ø–∏–¥–µ–º–∏–∏ —Å–æ–≥–ª–∞—Å–Ω–æ –º–æ–¥–µ–ª–∏ *SIR* —Å –ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º–∏, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—ã–º–∏ —Ä–∞–Ω–µ–µ:

–ö–∞–∫ –≤–∏–¥–∏–º, –ø–µ—Ä–µ–±–æ–ª–µ–ª–∏ –ø–æ—á—Ç–∏ –≤—Å–µ. –ü–æ–º–∏–º–æ –≥—Ä–∞—Ñ–∏–∫–æ–≤ —Ç–µ–∫—É—â–µ–≥–æ —á–∏—Å–ª–∞ –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö (*susceptible*), –∑–∞—Ä–∞–∂–µ–Ω–Ω—ã—Ö (*infectious*), –∏ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö (*removed*), –∑–¥–µ—Å—å –ø—Ä–∏–≤–µ–¥–µ–Ω –≥—Ä–∞—Ñ–∏–∫ —Ä–æ—Å—Ç–∞ –æ–±—â–µ–≥–æ —á–∏—Å–ª–∞ –ø–æ–¥—Ç–≤–µ—Ä–∂–¥—ë–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤ –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—è (*confirmed cases*), –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â–∏–π —Å–æ–±–æ–π –≤ –º–æ–¥–µ–ª–∏ *SIR* —Å—É–º–º—É —Ç–µ–∫—É—â–µ–≥–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –±–æ–ª–µ—é—â–∏—Ö –∏ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö (*confirmed cases = infectious + removed*). –ò–º–µ–Ω–Ω–æ —ç—Ç—É –∫—Ä–∏–≤—É—é —Ä–µ–≥—É–ª—è—Ä–Ω–æ –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –í–û–ó, —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Å–∞–π—Ç—ã —Ç–∏–ø–∞ [worldometer](https://www.worldometers.info/coronavirus/) –∏ —Ç—Ä–µ–≤–æ–∂–Ω—ã–µ —Å—Ç—Ä–∏–º—ã –Ω–∞ YouTube. –¢–∞–∫–∂–µ –∑–¥–µ—Å—å –ø—Ä–∏–≤–µ–¥–µ–Ω –≥—Ä–∞—Ñ–∏–∫ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–æ–π –ø–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤, —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏–∑—É—é—â–∏–π —Ç–µ–º–ø –∏—Ö —Ä–æ—Å—Ç–∞ (*rate of increase*). –ì—Ä–∞—Ñ–∏–∫ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö, –∫–∞–∫ —É–∂–µ —É–ø–æ–º–∏–Ω–∞–ª–æ—Å—å —Ä–∞–Ω–µ–µ, –≤–∫–ª—é—á–∞–µ—Ç –≤ —Å–µ–±—è –∫–∞–∫ –≤—ã–∑–¥–æ—Ä–æ–≤–µ–≤—à–∏—Ö (*recovered*), —Ç–∞–∫ –∏ —É–º–µ—Ä—à–∏—Ö (*deceased*):

–≠—Ç–æ –ø–µ—Ä–≤—ã–π —Å—Ç—Ä–µ—Å—Å-—Ç–µ—Å—Ç —Å–∏—Å—Ç–µ–º—ã –∑–¥—Ä–∞–≤–æ–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è. –ö–æ–ª–∏—á–µ—Å—Ç–≤–æ –Ω–æ–≤—ã—Ö –∑–∞—Ä–µ–≥–∏—Å—Ç—Ä–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤ –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—è –≤ —ç—Ç–æ—Ç –¥–µ–Ω—å –±—É–¥–µ—Ç –º–∞–∫—Å–∏–º–∞–ª—å–Ω—ã–º, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ –±–æ–ª—å–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã—Ö –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≥–æ—Å–ø–∏—Ç–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å, —Ç–æ–∂–µ –±—É–¥–µ—Ç –±–æ–ª—å—à–µ –æ–±—ã—á–Ω–æ–≥–æ, –∏ –≤—Å–µ–º –∏–∑ –Ω–∏—Ö –¥–æ–ª–∂–Ω–æ —Ö–≤–∞—Ç–∏—Ç—å –º–µ—Å—Ç.

–î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –¥–µ–Ω—å, –∫–æ–≥–¥–∞ —ç—Ç–∞ —Ç–æ—á–∫–∞ –±—É–¥–µ—Ç –ø—Ä–æ–π–¥–µ–Ω–∞, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≤—Å–ø–æ–º–Ω–∏—Ç—å —Ç–æ—Ç —Ñ–∞–∫—Ç, —á—Ç–æ –≤ —Ç–æ—á–∫–µ –ø–µ—Ä–µ–≥–∏–±–∞ –≤—Ç–æ—Ä–∞—è –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏–∏ $C(t)=I(t)+R(t)$, –æ–ø–∏—Å—ã–≤–∞—é—â–µ–π —Ç–µ–∫—É—â–µ–µ —á–∏—Å–ª–æ –ø–æ–¥—Ç–≤–µ—Ä–∂–¥—ë–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤, –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å —Ä–∞–≤–Ω–∞ –Ω—É–ª—é. –¢.–µ. –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–æ–¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞—Ç—å —Å—É–º–º—É –≤—Ç–æ—Ä–æ–≥–æ –∏ —Ç—Ä–µ—Ç—å–µ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏–π (–¥–ª—è $I$ –∏ $R$) –ø—Ä–∏–≤–µ–¥—ë–Ω–Ω–æ–π –≤—ã—à–µ —Å–∏—Å—Ç–µ–º—ã –î–£ *SIR*, –ø—Ä–∏—Ä–∞–≤–Ω—è—Ç—å –ø–æ–ª—É—á–µ–Ω–Ω—É—é –≤—Ç–æ—Ä—É—é –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é $C^{‚Ä≥}(t)$ –∫ –Ω—É–ª—é –∏ —Ä–µ—à–∏—Ç—å —ç—Ç–æ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ $t$:

$$
\begin{align}
& \frac{dC}{dt} = \frac{dI}{dt}+\frac{dR}{dt} = \frac{\beta IS}{N}, \\
& \frac{d^2C}{dt^2} = \frac{\beta}{N}\left(\frac{dI}{dt}S+\frac{dS}{dt}I\right) = 0, \\
& \frac{\beta IS^2}{N}-\gamma{IS}-\frac{\beta SI^2}{N} = 0, \\
& \frac{\beta IS}{N}\left(S-I-N\frac{\gamma}{\beta}\right) = 0, \\
& S - I = \frac{N}{R_0}.
\end{align} 
$$

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –ø–∏–∫ —Ç–µ–º–ø–∞ —Ä–æ—Å—Ç–∞ –ø–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤ (*rate of increase*) –¥–æ—Å—Ç–∏–≥–∞–µ—Ç—Å—è, –∫–æ–≥–¥–∞ —Ä–∞–∑–Ω–æ—Å—Ç—å —Ç–µ–∫—É—â–µ–≥–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –≤–æ—Å–ø—Ä–∏–∏–º—á–∏–≤—ã—Ö –∏ –∏–Ω—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∏–Ω–¥–∏–≤–∏–¥–æ–≤ —Å–æ—Å—Ç–∞–≤–ª—è–µ—Ç –≤–µ–ª–∏—á–∏–Ω—É $N/R_0$.

–í—Ç–æ—Ä–∞—è —Ç–æ—á–∫–∞ ‚Äì –º–∞–∫—Å–∏–º—É–º –∑–∞—Ä–∞–∂–µ–Ω–Ω—ã—Ö –Ω–∞ –¥–∞–Ω–Ω—ã—Ö –º–æ–º–µ–Ω—Ç –ª—é–¥–µ–π, –∏–ª–∏ –ø–∏–∫ —Ç–µ–∫—É—â–µ–≥–æ —á–∏—Å–ª–∞ –±–æ–ª–µ—é—â–∏—Ö, –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –ø—Ä–æ—â–µ. –î–ª—è —ç—Ç–æ–≥–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–∏—Ä–∞–≤–Ω—è—Ç—å –∫ –Ω—É–ª—é –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é $I'(t)$, —Ç.–µ. –≤—Ç–æ—Ä–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å–∏—Å—Ç–µ–º—ã –î–£ *SIR*:

$$
\begin{align}
& \frac{dI}{dt} = I\gamma\left(\frac{\beta}{\gamma}\frac{S}{N}-1\right) = 0, \\
& R_0\frac{S}{N}-1 = 0, \\
& S = \frac{N}{R_0}.
\end{align} 
$$

—Å–ª–µ–¥—É—é—â–∏–µ: $S(0)=N_{US}-1$, $I(0)=1$, $R(0)=0$. –ü–µ—Ä–≤–∞—è —Ñ—É–Ω–∫—Ü–∏—è `sir_model_2p` –º–æ–¥–µ–ª–∏ –∑–∞–¥–∞—ë—Ç —Å–∏—Å—Ç–µ–º—É –î–£ *SIR*, —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–Ω—É—é —Ä–∞–Ω–µ–µ. —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏—Ö –∫–æ–ª–∏—á–µ—Å—Ç–≤—É –∑–∞—Ä–∞–∂—ë–Ω–Ω—ã—Ö –∏ –ø–µ—Ä–µ–±–æ–ª–µ–≤—à–∏—Ö ‚Äì –ø–æ–¥—Ç–≤–µ—Ä–∂–¥—ë–Ω–Ω—ã—Ö —Å–ª—É—á–∞–µ–≤ –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏—è (*confirmed cases*) –Ω–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —á–∏—Å–ª–µ–Ω–Ω—ã–π –º–µ—Ç–æ–¥ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ `trf` (*[Trust Region](https://en.wikipedia.org/wiki/Trust_region) Reflective*), –±–∞–∑–∏—Ä—É—é—â–∏–π—Å—è –Ω–∞ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ —Ä–µ–≥–∏–æ–Ω–∞ –≤–æ–∫—Ä—É–≥ –ª—É—á—à–µ–≥–æ —Ä–µ—à–µ–Ω–∏—è, –≤ –∫–æ—Ç–æ—Ä–æ–º –∫–≤–∞–¥—Ä–∞—Ç–∏—á–Ω–∞—è –º–æ–¥–µ–ª—å –∞–ø–ø—Ä–æ–∫—Å–∏–º–∏—Ä—É–µ—Ç —Ü–µ–ª–µ–≤—É—é —Ñ—É–Ω–∫—Ü–∏—é.

{% highlight python linenos %}
Nus = 328.2e6 # 2019 US population

# SIR model ODE system
def sir_model_2p(y, t, beta, gamma): # 2 unknown parameters
    S, I, R = y # y[0]: susceptible, y[1]: infectious, y[2]: removed 
    dSdt = -beta * S * I / Nus
    dIdt =  beta * S * I / Nus - gamma * I
    dRdt =  gamma * I
    return dSdt, dIdt, dRdt

# Integrate a system of ODEs
def fit_odeint_2p(t, beta, gamma):
    # initial conditions (S(0)=Nus-1, I(0)=1, R(0)=0)
    ints = integrate.odeint(sir_model_2p, (Nus-1.0, 1.0, 0.0), t, args=(beta, gamma))
    return ints[:,1] + ints[:,2] # [:,1]: infectious + [:,2]: removed = confirmed cases

popt, pcov = optimize.curve_fit(fit_odeint_2p, t_data, y_data, bounds=([0,0], [np.inf,1]))
y_fitted = fit_odeint_2p(t_data, *popt)

def sir_model_3p(y, t, beta, gamma, pop): # 3 unknown parameters
    S, I, R = y 
    dSdt = -beta * S * I / pop
    dIdt =  beta * S * I / pop - gamma * I
    dRdt =  gamma * I
    return dSdt, dIdt, dRdt

def fit_odeint_3p(t, beta, gamma, pop):
    ints = integrate.odeint(sir_model_3p, (pop-1.0, 1.0, 0.0), t, args=(beta, gamma, pop))
    return ints[:,1] + ints[:,2]

popt, pcov = optimize.curve_fit(fit_odeint_3p, t_data, y_data, bounds=([0,0,y_data[-1]], [np.inf,1,Nus]))
